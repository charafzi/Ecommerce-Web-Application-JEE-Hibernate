## Overview
This web application is developed using Java Enterprise Edition (JEE) technologies and follows the Model-View-Controller (MVC) architecture.
The Ecommerce Web Application allows users to browse, search, and purchase products online. It provides features such as user authentication, product catalog management, shopping cart functionality, and order processing.

## Technologies Used

- Java Enterprise Edition (JEE): Servlets, JSP, JSTL
- Model-View-Controller (MVC) Architecture
- Hibernate to map object-oriented domain models to relational database
- Oracle as the relational database management system
- HTML, CSS, JavaScript and Boostrap for frontend development
- Apache Tomcat as application server
- Maven for project management and dependency resolution
